Cómo funciona el banco Arduino

La idea

Una placa Arduino lee sensores (entradas) y controla salidas (LEDs, buzzer, ventilador). Tu programa es el loop(): se repite todo el tiempo mientras la placa está encendida.

Sensores y salidas

  • Entradas analógicas (Luz, Temperatura, Potenciómetro): un valor de 0 a 100. Movelas con los sliders.
  • Entrada digital (Botón): presionado o suelto.
  • Salidas: encendido/apagado (LED rojo, LED verde, buzzer, ventilador).

Programar

Usá Si “sensor mayor/menor que valor” para reaccionar a una entrada, y Encender / Apagar una salida. ⚡ Encender pone a correr el loop; movés los sensores y ves la reacción en vivo.

Ojo: una salida queda como la dejaste hasta que la cambies (como en Arduino real). Por eso conviene usar el si no para apagarla cuando la condición deja de cumplirse.

Código real

El botón </> muestra el sketch de Arduino equivalente a tus bloques: el mismo programa, en el código que correría en una placa de verdad.

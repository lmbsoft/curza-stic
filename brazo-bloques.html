Cómo programar el brazo

La idea

Armás un programa apilando bloques, y el brazo lo ejecuta paso a paso. Cada bloque es una orden.

Construir

  • Tocá un bloque de la paleta para agregarlo a tu programa, en el punto que marca la barra blanca.
  • Tocá un bloque del programa (o el interior de un Repetir/Si) para elegir dónde se agrega el siguiente.
  • Con ↑ ↓ reordenás, con ✕ borrás. Los menús y números dentro del bloque cambian su parámetro.

Bloques de control

Repetir ejecuta lo de adentro varias veces. Repetir mientras sigue mientras se cumpla una condición. Si… si no… decide según lo que detecta el brazo.

Ejecutar

▶️ Ejecutar corre todo; ⏭️ Paso avanza un bloque por vez; ⏹️ detiene; ↺ vuelve todo al inicio. La tortuga/liebre regula la velocidad.

El desafío

Llevá las 3 plántulas de A a B. Pista: se puede hacer con un Repetir mientras “A tenga plantas” que adentro haga mover a A → agarrar → mover a B → soltar. 🌱

Sensor y clasificación

Cambiá la Tarea a Clasificar frutos: al agarrar un fruto, el sensor de la pinza detecta si está maduro. Usá Si “el fruto es maduro” { mover Maduros → soltar } si no { mover Verdes → soltar } dentro de un Repetir mientras “A tenga frutos”. Como vienen mezclados, el condicional es imprescindible. 🔴🟢
